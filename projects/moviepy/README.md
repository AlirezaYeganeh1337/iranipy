## Usage

Navigate into audio_video

```bash
cd audio_video
```

Install dependencies

```bash
poetry install
```

Run sample script

- poetry sample script

    ```bash
    poetry run sample
    ```

- Direct

    ```bash
    poetry run python audio_video/sample.py
    ```

You can find outputs in samples directory.
