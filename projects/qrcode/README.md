## Usage

Navigate into opencv_qrcode

```bash
cd opencv_qrcode
```

Install dependencies

```bash
poetry install
```

Run sample script

- poetry sample script

    ```bash
    poetry run sample
    ```

- Direct

    ```bash
    poetry run python opencv_qrcode/sample.py
    ```

You can find outputs in samples directory.
