## Usage

Navigate into poetry_demo

```bash
cd poetry_demo
```

Install dependencies

```bash
poetry install
```

Run main script

- poetry main script

    ```bash
    poetry run main
    ```

- Direct

    ```bash
    poetry run python poetry_demo/main.py
    ```
